# 🗺️ 山海行迹 - 智慧旅游指南网 (Travel Guide Website)

**山海行迹** 这是一个面向现代旅行者的高性能、一站式旅游指南平台。项目旨在整合中国城市、山水与海岛的真实旅行信息，告别虚假的 AI 绘图，全站采用来自公有领域的**真实网络图片**与**精准地理坐标**。通过构建互动的地图、多维度的筛选排序以及活跃的用户评论生态，帮助旅行者快速、直观地规划下一站行程。

---

## ✨ 核心功能与技术亮点

* **🖼️ 真实视觉复核**：拒绝 AI 虚假滤镜，全站切回互联网真实风光图片链接（如维基共享资源等公开学术/摄影渠道），真实还原目的地风貌。
* **🔍 智能搜索与多维筛选**：在目的地大厅支持按城市关键词、旅行标签（城市文化、海岛休闲、山水风光）进行实时动态检索，并支持按热度、评分、价格等条件进行前端高性能排序。
* **💬 动态互动评论生态**：内置独立的评论交互中心，支持实时评论提交、评分交互、列表动态渲染与本地持久化更新，大幅提升用户参与感。
* **🗺️ 本地坐标与位置服务**：项目集成地理坐标系统与可交互的本地地图组件，展示核心目的地的具体方位及周边景点，规划不绕路。
* **📱 极致的响应式布局**：基于移动优先（Mobile-First）设计理念重构全站样式，针对手机、平板、高分屏桌面端进行了深度流式适配，并包含完善的无障碍（ARIA）支持。

## 📂 项目文件结构

```text
├── index.html          # 网站主页（精品路线推荐、热门栏目、快捷搜索入口）
├── destinations.html   # 目的地大厅（全量卡片展示，支持多条件实时筛选与排序）
├── destination.html    # 目的地详情页（深度图文介绍、真实坐标与本地地图）
├── guide.html          # 旅游攻略与向导时间线
├── comments.html       # 独立用户评论中心
├── about.html          # 数据来源说明页（透明展示地理与图片信息出处）
├── app.js              # 核心业务逻辑引擎（驱动搜索、筛选、排序、评论和地图交互）
├── data.js             # 结构化静态数据集（旅游线路、真实图片库、坐标信息）
└── styles.css          # 全站统一响应式样式表（CSS3 Flexbox / Grid 现代布局）

```

## 🚀 本地开发与运行说明

### 1. 克隆/下载项目到本地

```bash
git clone git@github.com:chenxl6822/web_-.git
cd web_-

```

### 2. 启动本地服务

由于项目包含动态组件（JavaScript 模块化异步加载与数据驱动渲染），直接双击 HTML 文件可能会受到浏览器跨域或部分脚本限制。推荐使用本地 Web 服务器环境打开：

* **VS Code 用户（推荐）**：安装 `Live Server` 插件，在编辑器中右键 `index.html` 选择 **"Open with Live Server"**。
* **Python 用户**：在项目根目录下打开终端执行：
```bash
python -m http.server 8000

```


随后在浏览器中输入 `http://localhost:8000` 即可流畅访问。
